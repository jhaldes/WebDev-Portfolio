RC Flying Wing

Complete

This is my RC plane. I was able to finish it over a weekend, however it is still untested because I messed up one of the servo arms and I need to cut a new one. Being my first custom built plane, it is a kit from the company Flite Test that came with almost all of the electronics and all of the material to assemble the airfoil (the plane is a flying wing design so it is pretty much 100% airfoil).

Skills Used:
Model Building RC Electronics Aerodynamics

Custom Flight Controller

In Progress

This is the beginning of my custom flight controller. It is based off of the STM32 MCU, and will implement a gyroscope, barometer, and compass to maintain orientation (specific ICs are undetermined). This dev board is the beginning of this project. When completed, it will be able to stabilize an aircraft and provide data for autonomous navigation.

Technologies Used:
STM32 Embedded Systems PCB Design C Programming
